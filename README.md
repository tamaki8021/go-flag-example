### command

```
  $ ./cli-tool -h

  // サブコマンド
  $ ./cli-tool foo
  $ ./cli-tool bar

  $ ./cli-tool bar -h


  // フラグ
  $ ./cli-tool -version

  $ ./cli-tool -numb
  $ ./cli-tool -fork=true -numb=9
```
